# HEX-helper (英雄联盟海克斯大乱斗辅助工具)

一款面向《英雄联盟》极地大乱斗（ARAM）及海克斯大乱斗模式的实时数据辅助工具。

## 🌟 核心特性
- **数据自动进化**：应用启动后自动后台同步全量英雄胜率，确保数据永远保持最新赛季状态。
- **实时阶段感应**：自动切换视图。选人阶段显示“备战席胜率列表”，进入对局显示“锁定英雄攻略”。
- **精准海克斯分类**：内置本地映射库，100% 还原棱彩/金色/银色品阶，自动过滤无效数据。
- **智能出装推荐**：精准截取情境装备建议，内置常用装备 ID 翻译，彻底消灭 undefined。
- **高度自定义交互**：
    - **系统托盘**：支持最小化，右键进入设置。
    - **自由调节**：支持窗口透明度、UI 缩放（适配 4K 屏幕）。
    - **便捷操作**：支持全局快捷键 `Alt+V` 隐藏，全窗口无边框拖拽。

## 🛠 技术维护
- **数据存储**：用户配置与更新后的胜率库存储在 `%APPDATA%\HexHelper\`。
- **一键更新**：如需强制刷新数据，可在设置中点击“清理本地抓取缓存”。

## 📦 打包产物
打包产物位于 `dist/HexHelper.exe`，单文件绿色运行，无需安装。
