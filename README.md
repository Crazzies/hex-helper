# HEX-helper (英雄联盟海克斯大乱斗辅助工具)

一款面向《英雄联盟》极地大乱斗（ARAM）及海克斯大乱斗模式的实时数据辅助工具。基于 Electron 架构，深度集成 LCU 协议。

## 🌟 核心特性
- **实时阶段感应**：自动切换视图。选人阶段显示“备战席胜率列表”，进入对局显示“锁定英雄攻略”。
- **精准海克斯分类**：内置本地映射库（`augments_tier_map.json`），100% 还原棱彩/金色/银色品阶。
- **智能出装推荐**：自动过滤干扰项，精准截取第 10~21 个情境装备，内置常用装备 ID 翻译。
- **系统托盘运行**：支持最小化到托盘，右键可进入“应用设置”。
- **高度自定义**：支持窗口透明度调节、UI 缩放（适配 2K/4K 屏幕）、全窗口无边框拖拽。
- **故障回退机制**：抓取失败时自动切换至 Offline 离线模式，显示基础胜率参考。

## 🛠 维护说明
### 数据源与更新
- **英雄映射**：启动时自动从 LCU 同步，支持新英雄即时识别。
- **海克斯品阶**：由 `augments_tier_map.json` 静态驱动，如需大版本更新，可重新生成映射表。
- **抓取逻辑**：针对 `hextech.dtodo.cn` 动态 DOM 结构优化，支持胜率关键字精准拦截。

## ⌨️ 快捷键
- `Alt + V`: 全局显示/隐藏窗口。

## 📦 打包发布
```bash
npm run build
```
打包产物位于 `dist/HexHelper.exe`。
